Este es un nuevo borrador para probar la funcionalidad de branch y de paso recordar cosas importantes al respecto:

git branch borradores

git checkout borradores

/*operar con archivos y recordar que lo que agregue con add -a lo va a realizar dentro de esta rama*/

Continúo con el tradicional

git add -a

git status

git commit -m "neva rama borradores y .md con recordatorios"

Luego hay que asignarle por primera vez el origin a la rama, ej.

git push --set-upstream origin borradores

Esto va a permitir que la rama pueda usar el upstream origin.
